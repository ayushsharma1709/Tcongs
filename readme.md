# TCONGS

A modern, responsive redesign of the TCONGS website built with React. The project focuses on improving the user experience through a clean interface, responsive layouts, and reusable components.

## Features

- Modern UI with React
- Reusable component architecture
- Smooth navigation
- Optimized performance
- Clean and maintainable code

## Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- React Bootstrap

## Project Structure

```
src/
 ├── components/
 ├── pages/
 ├── assets/
 ├── App.jsx
 └── main.jsx
```

## Installation

```bash
git clone <repository-url>

cd tcongs-redesign

npm install

npm start
```
